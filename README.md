# WordPress Original Theme Portfolio

## Overview
オリジナルで制作したWordPressテーマです。  
静的サイトをWordPress化し、更新・運用を前提にテンプレート設計を行いました。

## Purpose
- WordPressのテンプレート階層の理解
- 固定ページ／投稿の使い分け
- 実務での運用を想定したテーマ構成の確認

## Technology
- WordPress
- PHP
- HTML / CSS / JavaScript

## Theme Structure
- front-page.php : トップページ用テンプレート
- page.php       : 固定ページ用テンプレート
- single.php     : 投稿詳細ページ
- archive.php    : 投稿一覧ページ
- index.php      : フォールバックテンプレート
- functions.php  : 機能追加・初期設定

## Key Points
- オリジナルテーマとして一から構築
- テンプレートの役割を明確に分離
- CSSはCMS運用を前提に設計
- 今後の拡張（カスタム投稿・決済機能等）を考慮した構造

## Environment
- Local development (XAMPP)
